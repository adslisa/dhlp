# DHgate Landing Page

这是一个为 DHgate 设计的推广落地页项目。

## 项目结构
- `index.html` - 主页面
- `styles.css` - 样式文件
- `script.js` - 交互脚本

## 功能特点
- 响应式设计，适配各种设备
- 现代化UI界面
- 突出展示DHgate的主要优势
- 清晰的行动召唤按钮
- 产品展示区域
- 用户评价展示

## 如何使用
1. 克隆此仓库
2. 在浏览器中打开 `index.html` 文件
3. 根据需要修改内容和样式

## 技术栈
- HTML5
- CSS3
- JavaScript
- Font Awesome 图标
- Google Fonts
